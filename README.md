# Movie Recommendation System

A simple Python-based movie recommendation system that suggests films based on user age and preferred genre. This project is built using Object-Oriented Programming (OOP) principles.

---

##  Features

- Add new movies to the system
- Filter movies by age restriction
- Recommend movies based on user genre preference
- Simple console-based interface

---

##  How It Works

The system stores a list of movies with:
- Name
- Genre
- Age restriction

Users enter their:
- Name
- Age
- Preferred movie genre

Then the system filters and recommends suitable movies.
